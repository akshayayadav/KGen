We present a semantic-based approach to multi-issue bilateral negotiation for e-commerce.
We use Description Logics to model relations among issues as axioms in a TBox.
We use Description Logics to model advertisements as axioms in a TBox.
We then introduce a logic-based alternating-offers protocol, able to handle conflicting information, that merges non-standard reasoning services in Description Logics with utility thoery to find the most suitable agreements.
